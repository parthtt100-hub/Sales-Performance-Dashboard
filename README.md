# Sales Profitability Dashboard
Overview

This project presents an interactive Excel dashboard developed to analyze sales performance and profitability across multiple business dimensions.

The objective was to transform raw transactional data into a structured, insight-driven reporting dashboard that supports performance evaluation and better decision-making.

The dashboard provides a clear summary of revenue trends, profit distribution, and category performance using pivot-based analysis and clean visual design.
Interactive Excel dashboard analyzing sales performance and profitability using pivot tables, charts, and structured business reporting techniques.
